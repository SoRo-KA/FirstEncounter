# The WEB API:

  This is a highlevel Python / REST API that you can use for scripting a scenario and use the different built-in robot functionalities and skills. It also ensures a minimum level of saftey and is the recommended API for most users. You can find details about it in the `STARTER_KIT`.

  The kit provides the following items:

  - Pymirokai documentation. Served it with python:
  ```
  python3 -m http.server -b 127.0.0.1 -d PATH/STARTER_KIT/DOCUMENTATION/pymirokai_sim_beta_documentation/html/
  ```
  - REST API documentation. Served it with python:
  ```
  python3 -m http.server -b 127.0.0.1 -d PATH/STARTER_KIT/DOCUMENTATION/REST_API_sim_beta_documentation/
